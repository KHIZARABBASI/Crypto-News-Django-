
# Crypto News
## Create Virtual Enviornment 
```cmd
py -m venv Env_Name
```

## steps to Follow

1. Activate Enviornment
```cmd
new_website\Scripts\activate.bat 
```

2. Change Diractkory
```cmd
cd crypto_news
```
3. Run Server
```cmd
python manage.py runserver
```
